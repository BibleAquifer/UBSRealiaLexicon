## 標題：指（finger） (id: REALIA:11.4.1.1)

11\.4\.1\.1 標題：指（finger）
========================

經文出處
----

Hebrew 來： אֶצְבַּע (音譯： ’etsba‘)

[JER 52:21](https://ref.ly/Jer52:21)

[JER 52:21](https://ref.ly/Jer52:21) ：在這節經文中，有些通俗譯本把原文字面意為「四指」的短語譯為「三英吋」（GNT (Good News Translation (1992)) 直譯）、「七厘米」（SPCL (Spanish Common Language Version (Dios Habla Hoy)) ）、「八厘米」（GECL (German Common Language Version (Gute Nachricht Bibel)) 、FRCL (French Common Language Version (Bible en français courant)) 、DUCL (Dutch Common Language Version) ）、「5厘米」（葡萄牙文通俗譯本），RSV (Revised Standard Version (1952)) 採用了直譯。較早、較莊重的譯本保留了原文的「四指」，甚至一些通俗譯本也這樣處理（ITCL (Italian Common Language Version) 、NOB ）。現代讀者讀到聖殿柱子的厚度為「四指」，也會與古時的讀者一樣浮現出一幅生動的畫面。

* **Associated Passages:** 耶利米書 52:21

* **Associated ACAI Concepts:** Finger (ID: `realia:Finger`)

## 標題：掌（palm） (id: REALIA:11.4.1.2)

11\.4\.1\.2 標題：掌（palm）
======================

經文出處
----

Hebrew 來： טֶפַח (音譯： tefach)

[1KI 7:26](https://ref.ly/1Kgs7:26), [2CH 4:5](https://ref.ly/2Chr4:5)

Hebrew 來： טֹפַח (音譯： tofach)

[EXO 25:25](https://ref.ly/Exod25:25), [EXO 37:12](https://ref.ly/Exod37:12), [EZK 40:5](https://ref.ly/Ezek40:5), [EZK 40:43](https://ref.ly/Ezek40:43), [EZK 43:13](https://ref.ly/Ezek43:13)

Hebrew 來： טַפְחָה (音譯： tafchah)

[PSA 39:6](https://ref.ly/Ps39:6)

希伯來文*tefach* /*tofach* 是手在指頭底部（不包括拇指）的寬度。

[EXO 25:25](https://ref.ly/Exod25:25); [EXO 37:12](https://ref.ly/Exod37:12); [1KI 7:26](https://ref.ly/1Kgs7:26); [2CH 4:5](https://ref.ly/2Chr4:5) ：許多語言可以像希伯來文本那樣表達，即「手的寬度」或「掌寬」。如果不能，則可以譯為「八厘米」（如FRCL (French Common Language Version (Bible en français courant)) ）或「三英吋」（如GNT (Good News Translation (1992)) ）。由於某種原因，CEV (Contemporary English Version) and NCV (New Century Version) 在《出埃及記》作「三英吋」（“three inches”），在《列王紀上》和《歷代志下》作「四英吋」（“four inches”）。

[EZK 40:5](https://ref.ly/Ezek40:5); [EZK 43:13](https://ref.ly/Ezek43:13) ：這裡提到的「肘」比平常的「肘」長一*tofach* 。這個量度是《以西結書》40–48章所用的量度。短肘的長度約為45厘米（18英吋），這裡的長肘應為53厘米（21英吋）。有些譯本想要反映出這個差異。然而，許多譯本將「短肘」和「長肘」都圓整為50厘米（18英吋）。

[PSA 39:6](https://ref.ly/Ps39:6) （《和》39:5）：這裡的*tefach* （*tapchah* ）一詞是一個象徵性的數字，表示人生短暫。耶穌在[MAT 6:27](https://ref.ly/Matt6:27) 所說的話可能就是源自這節經文。參下文關於「貝庫司」的討論。

* **Associated Passages:** 列王紀上 7:26; 歷代志下 4:5; 出埃及記 25:25; 出埃及記 37:12; 以西結書 40:5; 以西結書 40:43; 以西結書 43:13; 詩篇 39:6; 馬太福音 6:27

## 標題：肘、貝庫司（pēchus） (id: REALIA:11.4.2.1)

11\.4\.2\.1 標題：肘、貝庫司（pēchus）
============================

經文出處
----

Greek 希： πῆχυς (音譯： pēchus)

[MAT 6:27](https://ref.ly/Matt6:27), [LUK 12:25](https://ref.ly/Luke12:25), [JHN 21:8](https://ref.ly/John21:8), [REV 21:17](https://ref.ly/Rev21:17), [JDT 1:2](https://ref.ly/Jdt1:2), [JDT 1:2](https://ref.ly/Jdt1:2), [JDT 1:2](https://ref.ly/Jdt1:2), [JDT 1:2](https://ref.ly/Jdt1:2), [JDT 1:3](https://ref.ly/Jdt1:3), [JDT 1:3](https://ref.ly/Jdt1:3), [JDT 1:4](https://ref.ly/Jdt1:4), [JDT 1:4](https://ref.ly/Jdt1:4), [2MA 13:5](https://ref.ly/2Macc13:5), [1ES 6:24](https://ref.ly/1Esd6:24), [1ES 6:24](https://ref.ly/1Esd6:24), [S3Y 1:24](https://ref.ly/PrAzar1:24)

在新約中，希臘文*pēchus* （貝庫司）的長度等於舊約的「肘」（希伯來文*’amah* ）。

在大多數情況下，翻譯者可以取2貝庫司等於1米或1碼，不必使用累贅的分數或小數。

[MAT 6:27](https://ref.ly/Matt6:27); [LUK 12:25](https://ref.ly/Luke12:25) ：這裡的一個解經問題是，希臘文*pēchus* （貝庫司）通常指距離而不是時間。然而，這個詞也可以用在時間上，並且這個上下文指時間要合宜得多。另外，RSV (Revised Standard Version (1952)) 譯作“span of life”（「壽數」）的希臘文詞語更常用來衡量時間過於高度。現代譯本傾向於接受該詞衡量時間的意思。如果“span”（「持續時間、跨度」）用來衡量時間，那麼用*pēchus* 進行修飾表示時間很短，然而如果用來衡量人的身高，則是相當大的（約半米或18英吋）。[MAT 6:27](https://ref.ly/Matt6:27) 最後面的一句可以譯為：「使壽數多加一刻」（NAB (New American Bible (1970)) 直譯），「使壽命延長哪怕一天」（GECL (German Common Language Version (Gute Nachricht Bibel)) 直譯），或者「使壽數增加一點時間」，「活的再長一點點」，「使生命延長哪怕一點點」，等等。有些翻譯者會指出一個具體的時間，比如「甚至一天」、「甚至一小時」等。

[JHN 21:8](https://ref.ly/John21:8) ：這裡提到的距離約有100米或100碼。建議翻譯者參照CEV (Contemporary English Version) ，譯成「離岸只有大約一百碼」。

[REV 21:17](https://ref.ly/Rev21:17) ：經文指出尺寸為144肘，即12乘12。這似乎是一個象徵性的數字，許多譯本——包括GW (God's Word Translation) 、NCV (New Century Version) 、ITCL (Italian Common Language Version) 、GECL (German Common Language Version (Gute Nachricht Bibel)) 等通俗譯本——保留了原文的數字。ITCL (Italian Common Language Version) 在經文中還增加了公制的對應值，作「一百四十四肘［七十米］」。

GNT (Good News Translation (1992)) 認為144肘（72米或236英呎）是指城牆的高度（以及NJB (New Jerusalem Bible (1985)) 、REB (Revised English Bible (1989)) 、FRCL (French Common Language Version (Bible en français courant)) 、Phps (J.B. Phillips: The New Testament in Modern English (1958)) ）。然而，這裡也可能是指城牆的厚度（NIV (New International Version (1984)) 、BRCL (Brazilian Common Language Bible) ）。有些解經家指出，對於一座高1,500英里的城邑來說，城牆僅高72米（236英呎）顯得有些不相稱。但是也要指出，這裡的城牆不是用來保護城市（因為城門始終打開），而是用來分界。無論如何，人們支持「高度」勝過「厚度」。翻譯者應該選擇其中一個意見。如果像RSV (Revised Standard Version (1952)) 和其他一些譯本那樣僅僅說「一百四十四肘」（或「七十二米」、「二百三十六英呎」），而不指明這是高度，並不是一個合宜的翻譯。這節經文的前半節也可以譯成：「他又量了城牆［或譯：籬笆］的高度［或譯：厚度］。這個尺寸是七十二米［或譯：二百三十六英呎］。」

* **Associated Passages:** 馬太福音 6:27; 路加福音 12:25; 約翰福音 21:8; 啟示錄 21:17; 友弟德傳 1:2; 友弟德傳 1:3; 友弟德傳 1:4; 瑪加伯下 13:5; 厄斯德拉上 6:24; 三童歌 1:24

## 標題：英里（mile） (id: REALIA:11.4.2.5)

11\.4\.2\.5 標題：英里（mile）
=======================

經文出處
----

Greek 希： μίλιον (音譯： milion)

[MAT 5:41](https://ref.ly/Matt5:41)

[MAT 5:41](https://ref.ly/Matt5:41) ：這裡的距離有幾種表示方法。許多譯本保留了字面上的英里，甚至是在採用了公制單位的情況下。一般認為，「一英里」是羅馬士兵能夠強迫猶太人背著他的裝備行走的最長距離。這裡的要點不是確切的距離，而是「一英里」與「兩英里」之間的關係。即使在不知道英里的文化中，這個關係也會是顯然的，所以許多譯本譯為「一公里」和「兩公里」，以避免引入人們不知道的「英里」一詞。

事實上，這裡的距離可以替代為步行15–20分鐘所走的距離，甚至用步行時間來表達；比較林孔‧薩巴特克語（Rincon Zapotec）對於這節經文的翻譯：「如果有人強逼你背著他的行李走半個小時，你就同他走一個小時。」在有些文化中，按照「英里」在拉丁文中的原意來表達這個長度可能更為自然，即「一千步」。

* **Associated Passages:** 馬太福音 5:41

* **Associated ACAI Concepts:** Mile (ID: `realia:Mile`)

## 標題：Schoinos (id: REALIA:11.4.2.6)

11\.4\.2\.6 標題：Schoinos
=======================

經文出處
----

Greek 希： σχοῖνος (音譯： schoinos)

[2MA 11:5](https://ref.ly/2Macc11:5)

希臘文*schoinos* 表示的距離等於30斯他迪，約5\.6公里（3\.5英里）。[2MA 11:5](https://ref.ly/2Macc11:5) 有一個文本問題，似乎讀作「五斯他迪」，但這距離耶路撒冷太近了。在古時，文本就被校訂為「五*schoinos* 」。

* **Associated Passages:** 瑪加伯下 11:5
