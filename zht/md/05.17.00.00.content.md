## 標題：地毯、毯子（carpet, rug） (id: REALIA:5.17)

5\.17 標題：地毯、毯子（carpet, rug）
===========================

描述和用途
-----

![地毯](images/WEB-0489_carpet.jpg)

*地毯 (© Grand Parc \- Bordeaux, France, CC BY 2\.0, via Wikimedia Commons)*

地毯是鋪在房間地面上的物品。地毯可以用獸皮或布做成，布通常是用動物的毛織成的。

---

翻譯
--

[EZK 27:24](https://ref.ly/Ezek27:24) ：和本條目中的其他詞語一樣，希伯來文*genez* 一詞在聖經中可能僅出現一次。REB (Revised English Bible (1989)) 將其解作“stores”（「庫房」；參[EST 3:9](https://ref.ly/Esth3:9) 、[EST 4:7](https://ref.ly/Esth4:7) 和[3\.14\.1\.4 庫房、銀庫、寶庫 (treasury, strongroom)\<REALIA:3\.14\.1\.4\>](#) ），GECL (German Common Language Version (Gute Nachricht Bibel)) 作「衣服」，但大多數譯本都認為它的意思是「地毯」（“carpets”；RSV (Revised Standard Version (1952)) 、GNT (Good News Translation (1992)) ）或「小地毯」（“rugs”；NIV (New International Version (1984)) ）。

[JDG 5:10](https://ref.ly/Judg5:10) ：這裡的希伯來文*mad* 一詞的意思不詳，所以不同譯本的翻譯也很不同。其中一些譯法有：「華麗的毯子」、「毯子」（“rich carpet”、“carpets”；RSV (Revised Standard Version (1952)) 、NRSV (New Revised Standard Version (1989)) 、ITCL (Italian Common Language Version) 、GECL (German Common Language Version (Gute Nachricht Bibel)) 、SPCL (Spanish Common Language Version (Dios Habla Hoy)) 、路德、BE (Bible in Basic English) ）；「馬鞍」、「馬鞍墊子」、「馬鞍毯子」（“saddles”、“saddle\-cloths”、“saddle blankets”；GNT (Good News Translation (1992)) 、REB (Revised English Bible (1989)) 、NIV (New International Version (1984)) 、NCV (New Century Version) ）；「馬衣」（“caparisons”；JB (Jerusalem Bible (1966)) ）；「華麗的布」（“rich cloths”；猶太人出版學會聖經）、「審判」（“judgment”；《武加大譯本》、KJV (King James Version (1611)) ）。在這節經文中，發言者是在向許多百姓講話，但不清楚這些聽眾是被分成三個群體（RSV (Revised Standard Version (1952)) ，「你們騎黃褐色驢的，你們坐在華麗毯子上的，你們行走在路上的」），還是分成兩個群體（CEV (Contemporary English Version) ，「不論你是騎在備鞍的驢上，還是走路」）。聽眾被分爲幾個群體取決於希伯來文詞語*mad* 的意思。由於鞍子通常是由一塊很像地毯的布做成的，所以騎驢的人可能就是坐在毯子（做成的鞍子）上的人。

[ISA 21:5](https://ref.ly/Isa21:5) ：一些較早的譯本（如KJV (King James Version (1611)) ）把希伯來文*tsafith* 譯為「瞭望塔」。現在學者普遍認為，這個詞的意思可能是人們吃飯時坐著的地毯。

[JDG 4:18](https://ref.ly/Judg4:18) ：希伯來文*smikah* 在聖經中僅出現在此處。這個詞指一種厚布，可能是帳棚裡面的垂直掛布，或者用來禦寒的地毯。如果採取第一種解釋，本節經文的最後一個分句可譯為，「她把他藏在幔子後面」（如GNT (Good News Translation (1992)) ）；採取第二種解釋則可譯為，「她用毯子蓋住他」（如RSV (Revised Standard Version (1952)) ），或者像CEV (Contemporary English Version) 那樣增加一些隱含的信息，「［西西拉躺下來，然後］雅億用毯子將他蓋住。」
