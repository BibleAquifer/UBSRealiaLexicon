*埃及釘頭錘的頭，萊比錫大學古埃及博物館（Ägyptisches Museum der Universität Leipzig） (© Einsamer Schütze, via Wikimedia Commons)*

戰棒是一根比較短的木棍，在肉搏時用來擊打敵人。戰棒可以單手握持，擊打敵人身體的各個部位，尤其是那些沒有盔甲保護的部位。釘頭錘通常會裝上一個沉重的頭，用石頭或金屬製成。這個頭楔入棒的一端，用繩索綁牢或用棍棒的錐形來固定。握持部分也可能會逐漸變粗，從而使握持更牢靠。握柄有時會做成弧形。

翻譯
--

希伯來文*tothach* 只在[JOB 41:21](https://ref.ly/Job41:21) （《和》41:29）出現過一次，大部分學者同意這個詞指「棍棒」。有學者提出，這個詞也可能指箭的桿，不過我們查閱的譯本都沒有採用這種解釋。

在[EZK 9:2](https://ref.ly/Ezek9:2) ，重點是武器的可怕威力。有些譯本專注於表達這個意思，而不是試圖在目標語言中尋找一件準確對應的武器。因此，CEV (Contemporary English Version) 譯作“deadly weapon”（「致命的武器」），NCV (New Century Version) 作“powerful weapon”（「威力強大的武器」），FRCL (French Common Language Version (Bible en français courant)) 則譯成「毀滅性的武器」。

[EZK 39:9](https://ref.ly/Ezek39:9) ：很多譯本把這裡的希伯來文短語*maqel yad* （直譯：手棍、手杖）譯作「棍棒」（“clubs”；GNT (Good News Translation (1992)) ）或「戰棒」（“war clubs”；NCV (New Century Version) ）；但也有其他譯法，例如譯作「槍」（GECL (German Common Language Version (Gute Nachricht Bibel)) ）或「投槍」（SPCL (Spanish Common Language Version (Dios Habla Hoy)) ）。巴埃斯卡馬戈（G. Baez\-Camargo）提出，這是一種回力飛鏢（比較REB (Revised English Bible (1989)) 的“throwing\-sticks”「拋擲棒」，這個譯詞可以有幾種含意）。由於經文旨在表達將來會有大量木製武器被廢棄，並當作木柴來燒火，因此可以把這節經文列出的幾種兵器合併表達，歸在一個比較大的兵器類別之下；例如：ITCL (Italian Common Language Version) 把*maqel yad* 和「槍」合併為「矛槍」，CEV (Contemporary English Version) 把兩種盾一併稱為“shields”（「盾牌」）。

在[MAT 26:47](https://ref.ly/Matt26:47); [MAT 26:55](https://ref.ly/Matt26:55) 、[MRK 14:43](https://ref.ly/Mark14:43); [MRK 14:48](https://ref.ly/Mark14:48) 和[LUK 22:52](https://ref.ly/Luke22:52) 中，希臘文*xulos* 直譯作「木頭」，在這裡關於一群暴民行使私刑的語境中，這個詞可以簡單地理解為「棍子」。不過，我們查閱的所有譯本都把它譯作「棒」或同義詞語。
