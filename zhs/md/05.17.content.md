翻译
--

[EZK 27:24](https://ref.ly/Ezek27:24) ：和本条目中的其他词语一样，希伯来文*genez* 一词在圣经中可能仅出现一次。REB (Revised English Bible (1989)) 将其解作“stores”（“库房”；参[EST 3:9](https://ref.ly/Esth3:9) 、[EST 4:7](https://ref.ly/Esth4:7) 和[3\.14\.1\.4 库房、银库、宝库 (treasury, strongroom)\<REALIA:3\.14\.1\.4\>](#) ），GECL (German Common Language Version (Gute Nachricht Bibel)) 作“衣服”，但大多数译本都认为它的意思是“地毯”（“carpets”；RSV (Revised Standard Version (1952)) 、GNT (Good News Translation (1992)) ）或“小地毯”（“rugs”；NIV (New International Version (1984)) ）。

[JDG 5:10](https://ref.ly/Judg5:10) ：这里的希伯来文*mad* 一词的意思不详，所以不同译本的翻译也很不同。其中一些译法有：“华丽的毯子”、“毯子”（“rich carpet”、“carpets”；RSV (Revised Standard Version (1952)) 、NRSV (New Revised Standard Version (1989)) 、ITCL (Italian Common Language Version) 、GECL (German Common Language Version (Gute Nachricht Bibel)) 、SPCL (Spanish Common Language Version (Dios Habla Hoy)) 、路德、BE (Bible in Basic English) ）；“马鞍”、“马鞍垫子”、“马鞍毯子”（“saddles”、“saddle\-cloths”、“saddle blankets”；GNT (Good News Translation (1992)) 、REB (Revised English Bible (1989)) 、NIV (New International Version (1984)) 、NCV (New Century Version) ）；“马衣”（“caparisons”；JB (Jerusalem Bible (1966)) ）；“华丽的布”（“rich cloths”；犹太人出版学会圣经）、“审判”（“judgment”；《武加大译本》、KJV (King James Version (1611)) ）。在这节经文中，发言者是在向许多百姓讲话，但不清楚这些听众是被分成三个群体（RSV (Revised Standard Version (1952)) ，“你们骑黄褐色驴的，你们坐在华丽毯子上的，你们行走在路上的”），还是分成两个群体（CEV (Contemporary English Version) ，“不论你是骑在备鞍的驴上，还是走路”）。听众被分为几个群体取决于希伯来文词语*mad* 的意思。由于鞍子通常是由一块很像地毯的布做成的，所以骑驴的人可能就是坐在毯子（做成的鞍子）上的人。

[ISA 21:5](https://ref.ly/Isa21:5) ：一些较早的译本（如KJV (King James Version (1611)) ）把希伯来文*tsafith* 译为“瞭望塔”。现在学者普遍认为，这个词的意思可能是人们吃饭时坐着的地毯。

[JDG 4:18](https://ref.ly/Judg4:18) ：希伯来文*smikah* 在圣经中仅出现在此处。这个词指一种厚布，可能是帐棚里面的垂直挂布，或者用来御寒的地毯。如果采取第一种解释，本节经文的最后一个分句可译为，“她把他藏在幔子后面”（如GNT (Good News Translation (1992)) ）；采取第二种解释则可译为，“她用毯子盖住他”（如RSV (Revised Standard Version (1952)) ），或者像CEV (Contemporary English Version) 那样增加一些隐含的信息，“［西西拉躺下来，然后］雅亿用毯子将他盖住。”
