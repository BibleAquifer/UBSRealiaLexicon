## 标题：弦乐器、丝弦的乐器（stringed instruments） (id: REALIA:7.2)

7\.2 标题：弦乐器、丝弦的乐器（stringed instruments）
=======================================

![OAI-0062_stringed_instruments](images/OAI-0062_stringed_instruments.png)

*(Image generated by ChatGPT using OpenAI technology)*

许多乐器是把弦绷紧在共鸣箱上方，或连接到共鸣箱上，然后拨动弦使其振动，乐器便发出声音。

![OAI-0062_stringed_instruments](images/OAI-0062_stringed_instruments.png)

*(Image generated by ChatGPT using OpenAI technology)*

圣经中提到的各种弦乐器的辨识存在相当大的不确定性。另外，希伯来文*nevel* 和*kinor* 经常互换使用或平行出现（参[1SA 10:5](https://ref.ly/1Sam10:5) ；[2SA 6:5](https://ref.ly/2Sam6:5) ；[PSA 33:2](https://ref.ly/Ps33:2) ，[PSA 57:9](https://ref.ly/Ps57:9) ［《和》57:8］，[PSA 71:22](https://ref.ly/Ps71:22) ，[PSA 81:3](https://ref.ly/Ps81:3) ［《和》81:2］，[PSA 92:4](https://ref.ly/Ps92:4) ［《和》92:3］，[PSA 108:3](https://ref.ly/Ps108:3) ［《和》108:2］，[PSA 150:3](https://ref.ly/Ps150:3) 等），进一步增加了确认的难度。这两种乐器可能都是指里拉琴，只是大小不同，*kinor* 较小。我们很难明确区分*nevel* 和*kinor* ，两者主要的区别可能是共鸣箱的厚度，*nevel* 可能更厚一些。有关这些弦乐器的进一步讨论，参阅布朗（Braun）、拉威尔格伦（Lawergren，第55页）和奥康奈尔（O’Connell）的著作。

* **Associated Passages:** 撒母耳记上 10:5; 撒母耳记下 6:5; 诗篇 33:2; 诗篇 57:9; 诗篇 71:22; 诗篇 81:3; 诗篇 92:4; 诗篇 108:3; 诗篇 150:3
